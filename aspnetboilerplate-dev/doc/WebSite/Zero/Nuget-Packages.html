ASP.NET Boilerplate module-zero is distributed on nuget. Here, a list of all packages.

Shared

Shared packages between Abp.Zero.* and Abp.ZeroCore.* packages.

Abp.Zero.Common

Common package for both of Abp.Zero and Abp.ZeroCore libraries.

Abp.Zero.Ldap

LDAP authentication integration package.

Abp.Zero

Abp.Zero is built on Microsoft.AspNet.Identity.Core.

Abp.Zero

Core package of module zero.

Abp.Zero.Owin

Owin integration package for module zero.

Abp.Zero.AspNetCore

ASP.NET Core integration package for module zero.

Abp.Zero.EntityFramework

EntityFramework integration package for module zero.

Abp.ZeroCore Packages

Abp.ZeroCore is built on Microsoft.AspNetCore.Identity.

Abp.ZeroCore

Core package of module zero core.

Abp.ZeroCore.EntityFrameworkCore

Entity Framework Core integration package for module zero core.
